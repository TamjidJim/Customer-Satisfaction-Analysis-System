## Project Overview
A centralized platform for collecting, tracking and analyzing 
customer feedback across Agile software releases. The system 
helps Project Managers monitor satisfaction trends, Developers 
identify problem areas, and QA Analysts verify resolved issues.

## System Users
- **Project Manager (Admin):** Manages releases, assigns feedback, views reports
- **Developer:** Adds release details, updates feedback status
- **QA Analyst:** Verifies issue resolution, marks feedback as Verified
- **Customer:** Rates releases, submits feedback and suggestions

---

## Key Features
- Role-based user registration and login
- Software release creation and management
- Customer satisfaction rating and feedback submission
- Feedback workflow: Pending → Under Review → Resolved → Verified
- Analytical reports and satisfaction trend charts
