User Story:
As a Project Manager (Admin), I want to create
a role-based account with valid information so
that I can securely access the system and manage
all operations including releases, feedback, and reports.

Acceptance criteria:
- Form Includes: full name, email, username,
  password, confirm password, role = Project Manager
- Duplicate username or email shows error message
- After registration, redirected to Admin dashboard
- Password stored in encrypted form (Security NFR)
- Role-based access control enforced after registration
- System responds within 2-3 seconds (Performance NFR)
