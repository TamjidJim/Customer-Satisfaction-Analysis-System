Card ID   : BL-07
List      : Product Backlog
Title     : Receive alerts for critical feedback
Actor     : Developer
Colour     : Yellow/Orange

User Story:
As a Developer, I want to receive alerts for
critical feedback so that I can prioritize and
resolve urgent issues quickly.

Acceptance Criteria:
-- Alerts appear for low ratings and critical tags
-- Developer is notified immediately upon assignment
-- Alert includes feedback details and priority level
-- Alerts visible in developer dashboard
